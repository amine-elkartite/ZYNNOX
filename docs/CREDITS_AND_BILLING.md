# Credits And Billing

Credits are enforced server-side before paid AI actions.

Costs:

- AI chat: 1
- Web-search answer: 2
- AI Search quick: 3
- AI Search standard: 5
- AI Search deep: 8
- Landing page generation: 10
- Dashboard/admin generation: 15
- Full-stack generation: 30
- Code analysis: 2
- Security scan: 3

Billing defaults to demo mode. Demo mode simulates plan upgrades and credit purchases. Production mode uses Stripe-ready checkout and webhook validation.
